# 一、完善了一下博客页面，修复了个bug

## 总览，先来看你一下我再哪里改了把。

我在hexo博客的根目录_config.butterfly.yml里面的660行左右加上了如下的 🧩 代码吗

{% code lang: yaml%}

    # 开启懒加载（关键！）   #下面这一段坏了就删掉   好像成功了  不要乱删啦！！2025 12 16 22 56
    lazyload: true
    # 使用国内 CDN（加速）
    cdn: https://npm.elemecdn.com/twikoo@latest/dist/twikoo.all.min.js
    # 👇 关键：替换头像源为国内镜像
    avatar: https://cravatar.cn/avatar/%md5%?d=mp&s=160
{% endcode %}

之后还把butterfly主题目录下面的  _config.yml 里面的js代码给删了好像。然后  在根目录终端运行 

{% code lang: %}
npm run dev
{% endcode %}

## 加了这段代码的效果

### 效果当然就是，加上这些修改之后，那个留言板就可以打开的快了，lazy加载也整上了。整体来说，还是很不错的。
