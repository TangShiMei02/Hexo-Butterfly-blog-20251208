# 在 Hexo + Butterfly 主题博客中使用彩色文字

#### 写这个是为了让你别忘了咋写彩字儿。
#### <span style="color: #ff6b6b;">关于如何</span>在 Hexo 博客（Butterfly主题）的文章中添加带颜色的文字。最简单的方法：直接在 Markdown 中使用 HTML"<span>"标签。

## 通用方法：使用 `<span>` 标签

### 这是最直接、可靠的方式，不需要额外配置。

## <span style="color: rgba(255, 0, 0, 1)">示例👇</span>

---

## 指定几大原色(红色文字)
{% code lang:text %}
<span style="color: red;">这段是红色文字</span>
{% endcode %}

---

## 自定义颜色
{% code lang:text %}
<span style="color: #ff6b6b;">这段是珊瑚红文字</span>
{% endcode %}

---

## RGB颜色
{% code lang:text %}
<span style="color: rgba(0, 128, 255, 1);">这段是蓝色文字</span>
{% endcode %}
---

## 使用方法
在你的文章中任何位置，都可以直接插入上面的代码片段来显示不同颜色的文字。

例如：

{% code lang:text %}
正常文字，<span style="color: red;">这是红色文字</span>，然后继续。
{% endcode %}
